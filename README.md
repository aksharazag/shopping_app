# Flutter shopping_app
A Flutter shopping application built using Clean Architecture and GetX. The project demonstrates scalable architecture, local data persistance using Hive and common e-commerce features.

## Features

- User Login
- User Registration
- Home screen
- Product Listing
- Add to Cart
- Remove from cart
- quantity
- Local storage using Hive

## Architecture

Clean Architecture - domain layer, data layer, presentation layer

## Tech Stack

- Flutter
- Dart
- GetX
- Clean Architecture
- Rerpository Pattern

## State Management

- GetX

## Local Database

-Hive
